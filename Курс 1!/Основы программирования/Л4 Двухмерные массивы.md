Двухмерный массив представляет собой прямоугольную таблицу, состоящую из строк и столбцов. Доступ к элементу производится по 2 индексам и имени массива.

##### Описание
```
тип имя[][]; // [индекс строки][индекс столбца]
```
Работа с двухмерным массивом аналогична работе с одномерным, но требует хотя бы двойного цикла.

Чтобы вывести двухмерный массив можно использовать `cout.width()`, который зарезервирует количество символов под каждое число.
